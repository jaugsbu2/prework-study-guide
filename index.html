Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The head element can contribute to Search Engine hits.
  • The image src is the location to pull the image from.
  • There can be up to 6 heading levels.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A class selector can be used to assign properties to specific sections.
  • A border is located between the padding and margin and can be colored.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls the latest code for the main branch
  • Need to pull origin after creating a branch
  • Code . opens Visual Studio in the branch shown on cmd
